1. What is the difference between an adaptive website and a fully responsive website?

 Adaptive websites adapts itself to various screen sizes by using media queries and relies on predefined screen sizes. Responsive design combines features from fixed, adaptive, and fluid layouts
 while using media queries and loads faster due to using one master layout to be loaded to various devices.

2. Describe what it means to be mobile first vs desktop first.

  Mobile first is designing for a mobile device first then expanding features and layouts as you gain more space.
  Whereas desktop first implies that you are starting from a larger screen and working your way down to smaller screens adjusting features and layouts
  using media queries.
 
3. What does `font-size: 62.5%` in the `html` tag do for us when using `rem` units?

  62.5% equates to 10 pixels, therefore, making it easier to do math and using rem.
